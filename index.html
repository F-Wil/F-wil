Welcome

F-Wil provides an interactive map of Hamburg's transit infrastructure, including bus depots (Betriebshöfe) and turnaround loops (Wendeschleifen). This project visualizes geographic data about the city's public transportation network.

Features

  • 🗺️ Interactive map powered by Leaflet and OpenStreetMap
  • 🔍 Filter by location type (Depots, Turnarounds)
  • 📍 Detailed location information on click
  • 🌍 Centered on Hamburg, Germany

Getting Started

Click here to access the interactive map. Use the filter controls to show or hide specific types of transit infrastructure. Click on any location to see more details.
